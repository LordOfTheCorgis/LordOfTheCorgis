### Lumix Solutions

Infrastructure built by engineers. Reliable compute, game server hosting, voice infrastructure, and enterprise-grade DDoS protection, built by people from sysadmin, network engineering, and software backgrounds who wanted something they'd actually use themselves.

- **Game Server Hosting** — low-latency networking, included DDoS protection, one-click deploy, and automatic backups (FiveM and more)
- **Voice Infrastructure** — TeamSpeak &amp; Mumble with priority traffic routing, a low-latency codec, and custom branding
- **Application Hosting** — managed Node.js, Python &amp; Docker with push-to-deploy, auto-scaling, and zero-downtime updates

`99.99% SLA` · `10+ Tbps mitigated` · `sub-10ms latency` · `12 points of presence`
